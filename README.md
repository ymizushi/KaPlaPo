KaPlaPo
================================

PlanningPokerアプリです。

仕事でよくPlannningPokerをするのですが、
自分で好きな画像とか音楽設定したかったので自作しました。

もしかしたら、色々UIとか変えて更新とかもするかも

# 実装済
- 丸い円を選択して、指定の画像を表示する機能

# 未実装アイデア
- 表示する画像を設定出来る機能（画像リサイズはあらかじめ、おこなってもらう必要がある）
- 表示する画像をリサイズして、設定出来る機能
- データベースに出したカウントと実際に取られたカウントを投げる機能
- IRC通知機能
- 画像を格好よく出す
